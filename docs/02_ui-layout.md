# 02. UIレイアウト・共通コンポーネント

## 概要

アプリケーション全体のレイアウトと共通UIコンポーネントを作成する。

## 完了条件

- 共通レイアウト（ヘッダー、ナビゲーション）が実装されている
- 動画作成画面の3カラムレイアウトが実装されている
- 必要なshadcn/uiコンポーネントが導入されている

## 技術仕様

| 項目 | 内容 |
|------|------|
| UIライブラリ | shadcn/ui |
| テーマ | 明るめダーク系（柔らかいダークグレー） |
| レスポンシブ | PC優先（MVP段階ではスマホ非対応） |

## タスク

- [x] 共通レイアウトコンポーネント作成
- [x] ヘッダーコンポーネント（ロゴ、ユーザーメニュー）
- [x] 動画作成画面の3カラムレイアウト
  - 左: ステップナビゲーション
  - 中央: 編集エリア
  - 右: プレビューエリア
- [x] 必要なshadcn/uiコンポーネント導入
  - Button, Input, Textarea
  - Card, Dialog, Toast (Sonner)
  - Select, Tabs
  - Progress, Skeleton
- [x] ローディング・エラー状態のUI

```
│ ⑤⑥    │                       │               │

```
