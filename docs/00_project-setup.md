# 00. プロジェクト初期設定

## 概要

Next.js 15プロジェクトの基盤設定を行う。shadcn/ui導入、Supabase接続、環境変数設定など。

## 完了条件

- shadcn/uiが導入され、コンポーネントが使用可能
- Supabaseプロジェクトと接続完了
- 環境変数が適切に設定されている
- 開発サーバーが正常に起動する

## 技術仕様

| 項目 | 内容 |
|------|------|
| UIライブラリ | shadcn/ui |
| バックエンド | Supabase（認証・DB・Storage） |
| デザインテーマ | ダークモード（柔らかいダークグレー） |
